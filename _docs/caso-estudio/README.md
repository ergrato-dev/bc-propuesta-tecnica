## Nota para Instructores

El caso FerreMax está diseñado para ser **consistente y progresivo**. Cada semana se agrega un documento nuevo al caso que representa el trabajo que haría un consultor real en esa etapa del proyecto. Los aprendices trabajan con el caso en teoría y prácticas, no en sus entregables personales.

**Regla importante:** nunca modificar un documento de una semana anterior una vez que la semana haya sido dictada — el caso debe ser estable como referencia.
