## 🔗 Referencias en Línea

### Herramientas Gratuitas

| Herramienta | URL | Para qué usarla |
|-------------|-----|----------------|
| **ProjectLibre** | https://www.projectlibre.com | Gantt profesional con gestión de recursos — alternativa open source a MS Project |
| **ClickUp (free)** | https://clickup.com | Gestión de tareas con vista de Gantt y seguimiento de tiempo |
| **GitHub Projects** | https://github.com/features/project-management | Tablero Kanban y vista de roadmap integrada al repositorio |
| **Miro (free tier)** | https://miro.com | Diagramas colaborativos y tableros de cronograma visuales |
| **draw.io / diagrams.net** | https://www.diagrams.net | Diagramas de flujo, dependencias y cronogramas — 100% gratuito |
