## 📺 Videos Recomendados

### 1. ¿Qué es el Diagrama de Gantt? — Explicación visual paso a paso
- **Duración**: ~10 minutos
- **Contenido**: Qué es un Gantt, cómo se lee, cuándo usarlo

### 2. Cómo hacer un Gantt en Google Sheets (gratis, sin herramientas de pago)
- **Duración**: ~15 minutos
- **Contenido**: Tutorial paso a paso para construir un Gantt simple con formato condicional
- **Búsqueda sugerida**: `"gantt google sheets tutorial español gratuito"`
- **Por qué verlo**: Google Sheets es una herramienta que todos tienen — no requiere instalación

- **Búsqueda sugerida**: `"projectlibre tutorial español"`
- **Por qué verlo**: ProjectLibre es la herramienta Open Source alternativa a MS Project — 100% gratuita

### 4. Velocidad del Equipo en Scrum — ¿Qué es la velocidad y cómo se calcula?

- **Búsqueda sugerida**: `"velocidad equipo scrum story points español"`
- **Por qué verlo**: Complementa la Teoría 02 de esta semana

### 5. La Ley de Brooks explicada — Por qué más programadores no resuelven el retraso
- **Duración**: ~12 minutos
- **Contenido**: El error clásico de agregar equipo a un proyecto tardío
- **Búsqueda sugerida**: `"ley de brooks software proyectos retrasos español"`
