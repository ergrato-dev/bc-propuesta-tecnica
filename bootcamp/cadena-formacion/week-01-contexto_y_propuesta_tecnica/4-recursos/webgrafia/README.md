## Glosario de Términos del Sector TI Colombiano

| Término | Qué significa en contexto laboral colombiano |
|---------|---------------------------------------------|
| **Propuesta** | Documento que envías al cliente antes de firmar |
| **Negocio cerrado** | Cuando el cliente aprueba la propuesta y firma |
| **Outsourcing** | Contratar equipo externo para el desarrollo |
| **Ficha técnica** | Resumen de especificaciones de un sistema |
| **Requerimiento** | Sinónimo local de "requisito" — muy usado en Colombia |
| **Mesa de ayuda / soporte** | Servicio de atención post-entrega |
