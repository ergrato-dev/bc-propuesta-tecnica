## Vocabulario del Sector

> En Colombia el término "requerimiento" es tan frecuente como "requisito". Ambos son válidos, aunque el estándar internacional usa "requisito" (IEEE) o "requirement" (inglés).

| Término en Colombia | Equivalente estándar | Contexto de uso |
|--------------------|---------------------|-----------------|
| **Requerimiento** | Requisito | Usado en la mayoría de empresas colombianas de software |
| **Levantamiento** | Elicitación / Relevamiento | "Levantar requisitos" = hablar con el cliente para obtener información |
| **Acta de entendimiento** | Acta de reunión | Documento que formaliza lo acordado con el cliente |
| **Historia de usuario** | User Story | Formato ágil de requisito funcional: "Como [rol] quiero [acción] para [beneficio]" |
| **Caso de uso** | Use Case | Descripción de cómo un actor interactúa con el sistema para lograr un objetivo |
